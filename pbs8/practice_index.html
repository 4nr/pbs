Positioned Boxes

  • static: normal flow, unpositioned
  • fixed: fixed in relation to the browser window
    1. top: pixels down
    2. left: pixels in
    3. right: pixels in
    4. bottom: pixels up
    5. width, height: pixels or auto
  • absolute: defined in relation to its nearest positioned parent
  • relative: offset from normal position by values you define. can be used simply to make box "positioned"
  • z-index: can be used to specify layer, think stack of projector transparency sheets - bigger numbers on top
