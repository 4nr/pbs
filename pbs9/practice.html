CSS Layout Properties

Review

Float
float: none (default) | left | right; property is declared and tag is removed from the normal flow, tags above and below move to meet, floated tag swims in lower content.
Clear
clear: none (default) | left | right | both; property on a tag declares whether a tag from above is prohibited from penetrating into the top of it. ie: no prohibitions | left half | right half | both halves prohibited.
Position
position: static (default) | fixed | absolute | relative;
static - not considered positioned, just normal flow
fixed - out of normal flow, defined in relation to browser window
absolute - out of normal flow, defined in relation to nearest positioned parent
relative - place-holder remains in normal flow, optionally specify offset
z-index
think in terms of layering. for unpositioned (static) boxes simply nest tags, for positioned elements specifying a z-index: [whole number] will stack tags like sheets of paper with larger numbers in front.
Display
tags that are normally block | inline | inline-block can be switched to another using display: [value];or vanished using display: none;
create a flex box using display: flex; and its immediate children will become flex items and displayed as columns
a child element can have flex: [1|2|3|etc]; to specify its width relative to siblings, and order: [1|2|3|etc]; to specify appearance order left to right.
