PBS 5: Example 2 - Links & Anchors

Absolute Links

Links can contain absolute URLs, like this one to my home page. Notice that that link opens my home page in a new window/tab.

Relative Links

Relative Links comes in many different forms.

Relative links can be relative to the host, for example, assuming you didn't delete the files, I should be able to link relatively to the example from the first installment - PBS1 Example.

Relative links can also be relative to the path, for example, this link goes to another file in the same folder.

Relative links can also point to fragments or anchors within the current page. I've included anchors at the start of each section in this page, so the following relative link will jump your browser down just a tiny bit to the section on Anchors & Fragments.

Fragments & Anchors

Imagine this is a big report about something. It will be broken into sections, and users would probably like an index at the start of the document, and the would like to be abel to click on the names of the sections to jump to them. This can be done with named fragments/anchors.
